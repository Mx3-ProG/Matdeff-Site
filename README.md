# Matdeff Site

Ce dépôt contient le site statique de `Matdeff Site`, conçu pour présenter une landing page, un tableau de bord et un formulaire d’inscription.

## Structure du projet

- `landing-page.html` : page d’accueil / landing page principale.
- `landing-style.css` : styles CSS associés à la page de présentation.
- `dashboard.html` : page de tableau de bord (dashboard).
- `signup.html` : page ou formulaire d’inscription.
- `.gitignore` : fichiers et dossiers à ignorer par Git.
- `README.md` : description du projet et de la structure.

## Objectif

Ce projet vise à fournir une base simple et statique pour une présentation de service ou d’application. Il peut être déployé sur GitHub Pages ou tout autre hébergement de sites statiques.

## Utilisation

1. Ouvrir les fichiers HTML dans un navigateur pour visualiser le site.
2. Modifier les fichiers HTML/CSS selon les besoins.
3. Pousser les changements dans le dépôt GitHub pour versionner le projet.
